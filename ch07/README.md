## 要点  

■　声明一个映射变量的时候，你必须提供键和值的类型：  

```go
var myMap map[string]int
```

■　为了创建一个新的映射，调用make函数并传入需要创建的映射的类型：  

```go
myMap = make(map[string]int)
```

■　为了将一个值传入映射，需要提供一个对应的键并用方括号括起来：  

```go
myMap["my key"] = 12
```
